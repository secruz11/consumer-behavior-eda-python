## Consumer Behavior & Engagement Analysis

This project explores consumer shopping behavior using Python, with a focus on behavioral patterns, engagement trends, and purchasing decisions.

### Overview
The analysis includes data cleaning, exploratory data analysis (EDA), visualization, and regression modeling to understand factors influencing purchase amount and consumer behavior.

### Tools & Methods
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Exploratory Data Analysis (EDA)

### Key Focus Areas
- Behavioral and engagement patterns
- Feature-driven insights into purchasing behavior
- Data-driven interpretation of consumer data

### Dataset
Consumer Behavior and Shopping Habits dataset sourced from Kaggle.
